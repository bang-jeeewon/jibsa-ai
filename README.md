## jibsa-ai 🚀

청약 공고를 자동으로 읽고, 일정 관리부터 질문 답변까지 한 번에 처리하는 RAG 기반 웹 앱입니다.  
배포 링크: https://jibsa-ai.onrender.com

### 한눈에 보는 하이라이트 📌

- **원클릭 분석**: 캘린더에서 공고 선택 → PDF 다운로드·정제 → 질의응답까지 자동 진행
- **문서 범위 보장**: 공고별로 벡터 DB를 분리해 질문이 해당 문서 안에서만 답변되도록 설계
- **모델 선택**: GPT-4o-mini 또는 Gemini Pro 중 원하는 LLM을 선택해 답변 생성
- **운영 편의**: DB 리셋 기능으로 잘못된 데이터 적재 시 즉시 초기화

### 사용 방법

1. 상단 링크로 접속 후 캘린더에서 관심 있는 공고를 클릭합니다.
2. “분석”을 누르면 공고 PDF를 자동으로 내려받아 학습합니다.
3. 하단 챗 영역에 궁금한 점을 입력하면 AI가 해당 공고 내용을 바탕으로 답변합니다.
4. 데이터가 꼬였을 땐 “DB 리셋”을 눌러 초기화하세요.

### 주요 기능

- 공고 일정 캘린더 뷰 & 상세 메타데이터 확인
- PDF 텍스트/표 추출 후 마크다운 변환 및 청킹
- 벡터 DB 기반 문서 한정 질의응답 (RAG)
- 모델 스위칭: OpenAI ↔︎ Gemini
- Render 배포 환경 최적화(임시 파일 정리, 경로 보정)

### 기술 스택 (Python 중심)

- **웹**: Flask + Jinja 템플릿
- **RAG 파이프라인**: PDFExtractor → DataProcessor(텍스트/표 정제) → TextChunker → Chroma(VectorStoreService)
- **임베딩/LLM**: OpenAI text-embedding-3-small, GPT-4o-mini / Google Gemini Pro
- **데이터**: Chroma 퍼시스턴스, 공고 크롤링 & PDF 다운로드 서비스 연동
- **배포**: Render (환경 변수 기반 경로 설정, 임시 파일 정리 로직 포함)

### 왜 만들었나요?

청약 공고 PDF는 표와 특수 서식이 많아 빠르게 읽기 어렵습니다. 자동 추출·정제 후 RAG로 질문에 바로 답하게 하여, 실무자나 예비 청약자가 필요한 정보를 즉시 얻도록 돕기 위해 만들었습니다.
